Virgil is a simplified mobile interface with customizeable options that let you display only the essential features you need.
Virgil is designed for non-smartphone people. Whether child, elderly, special needs, or just technologically challenged, Virgil can help make you and your' family's lives simpler, easier, and safer.

Simple

Core features from your phone simplified into a clean, large, easy to read user interface. No more fumbling with tiny keyboards. Everything you need is directly accessible.

Essential

No more smartphone overload: Virgil lets you customize clear, concise options people can understand, and gives you only the essential features you need.

Universal

Virgil works with all major smartphones. To fully utilize all features, we recommend a phone with the latest web browsers and geolocation.
